# POC_and_EXP

本项目收录各类漏洞的概念验证（POC）代码与利用（EXP）脚本，涵盖 Web 应用、中间件、CMS、服务组件等多元化场景。旨在为网络安全研究、漏洞原理学习以及 CTF 竞赛备考提供实战化的代码参考。

## 📌 项目目标
- **漏洞原理学习**：通过分析实际代码，深入理解漏洞的触发机制与利用过程。
- **CTF 备考**：提供经过验证的利用脚本，作为网络安全爱好者实战演练的参考工具。
- **授权安全测试**：辅助白帽子安全研究员进行合法合规的授权渗透测试与漏洞验证。

## 📂 目录结构 (规划)
本项目按照漏洞类型或目标组件进行分类存放，具体结构规划如下：

```text
POC_and_EXP/
├── Web/                 # Web应用通用漏洞 (如 SQLi, XSS, SSRF, 文件上传等)
├── Middleware/          # 中间件组件漏洞 (如 Tomcat, Nginx, WebLogic, JBoss 等)
├── CMS/                 # 知名 CMS 系统漏洞 (如 WordPress, ThinkPHP, FastAdmin, DedeCMS 等)
├── CVE/                 # 按照 CVE 编号整理的专项漏洞利用脚本
├── .gitignore           # Git 忽略文件配置
└── README.md            # 项目说明文档
```

## 🤝 贡献指南
